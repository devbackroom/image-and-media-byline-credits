This plugin adds a byline credit to images.

== Description ==
This plugin adds a byline credit to images.

* Creates a Photographer user role
* Changes image description field label to include \"Image Byline\"
* Autocompletes with the photographer list in the image description
* Wraps the photographer name in a byline shortcode to distinguish it from text
* Appends the byline/description to the image caption

== Installation ==
1. Upload `image-byline.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the \"Plugins\" menu in WordPress
